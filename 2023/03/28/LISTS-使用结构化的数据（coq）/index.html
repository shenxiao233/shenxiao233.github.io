构造和操作列表的函数

Repeat

第一个是 repeat 函数，它接受一个数字 n 和一个 count，返回一个长度为 count，每个元素都是 n 的列表。

Length

length 函数用来计算列表的长度。

Append

app 函数用来把两个列表联接起来。

由于下文中随处可见，不妨将其定义为中缀运算符。

Head 与 Tail

下面介绍列表上的两种运算：hd 函数返回列表的第一个元素（即“表头”）； tl 函数返回列表除去第一个元素以外的部分（即“表尾”）。由于空表没有表头， 我们必须传入一个参数作为返回的默认值。
